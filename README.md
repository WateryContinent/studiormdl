> [!CAUTION]

## Features
- **RUI Meshes:** Allows for RUI Meshes to be compiled into models
- **Complete Compilation:** Compiles straight to RMDL
- **Animation Conversion:** **StudioRMDL** can convert animations from the SMD into rseq and rig
